Quick Start

Installation

pip install -r requirements.txt

# For real image downloading (optional)
pip install pystac-client planetary-computer

Basic Usage

import numpy as np
import cv2
from pathlib import Path

# Load satellite image
image_path = Path('data/images/sample_001.png')
image = cv2.imread(str(image_path))
print(f"Image shape: {image.shape}")

Advanced Features Example

from advanced_processing import AdvancedImageProcessor
from ml_integration import SatelliteMLProcessor
from enhanced_real_image_downloader import EnhancedRealImageDownloader

# Advanced processing
processor = AdvancedImageProcessor()
edges = processor.detect_edges(image, method='canny')
features = processor.extract_all_features(image)

# ML integration
ml = SatelliteMLProcessor()
buildings = ml.detect_buildings_simple(image)
ndvi = ml.extract_ndvi(image)

# Download real images
downloader = EnhancedRealImageDownloader()
files = downloader.download_sample_real_images(num_images=10)

Run Complete Demo

python advanced_example.py

Complete Documentation

Project Overview

This satellite image dataset project provides comprehensive tools for working with satellite imagery, including:

  • Data Loading: Load images, labels, metadata, and building detections
  • Image Processing: Basic and advanced image processing capabilities
  • Machine Learning: Classification, detection, and feature extraction
  • Visualization: Statistical analysis, 3D plots, and interactive dashboards
  • Real Data: Download real satellite images from multiple sources

Advanced Features Details

Advanced Processing
  • Edge Detection (Canny, Sobel, Laplacian)
  • Image Segmentation (Watershed, SLIC)
  • Feature Extraction (HOG, LBP, GLCM)
  • Image Enhancement (CLAHE, Histogram EQ)
  • Noise Reduction (Bilateral, NLM)
ML Integration
  • Land Cover Classification
  • Building Detection
  • Change Detection
  • NDVI Extraction
  • Training Dataset Creation
Real Image Download
  • Planetary Computer (No credentials)
  • USGS EarthExplorer (Landsat)
  • Copernicus Hub (Sentinel-2)
  • Multiple locations worldwide
  • Automatic metadata extraction
Visualization
  • 3D Surface Plots
  • Statistical Analysis
  • Comparison Views
  • Time Series Visualization
  • Interactive Dashboards

Installation & Troubleshooting

Required Packages:
pip install numpy opencv-python matplotlib scikit-image scikit-learn scipy seaborn tqdm rasterio pillow
Optional (for real images):
pip install pystac-client planetary-computer

Check Installation: Run python check_errors.py to verify all packages are installed correctly.

Download Real Images

To download real satellite images:

from enhanced_real_image_downloader import EnhancedRealImageDownloader

downloader = EnhancedRealImageDownloader()
files = downloader.download_sample_real_images(num_images=10)

Note: Requires internet connection and optional packages (pystac-client, planetary-computer)
